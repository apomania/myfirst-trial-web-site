WEB이란 무엇인가?

일반적으로 웹으로 알려진 WWW(World Wide Web)는 인터넷을 통해 문서 및 기타 웹 리소스에 액세스할 수 있도록 하는 정보 시스템입니다.

문서 및 다운로드 가능한 미디어는 웹 서버를 통해 네트워크에서 사용할 수 있으며 웹 브라우저와 같은 프로그램에서 액세스할 수 있습니다. World Wide Web의 서버와 리소스는 URL(Uniform Resource Locator)이라는 문자열을 통해 식별되고 배치됩니다. 원본과 여전히 매우 일반적인 문서 유형은 하이퍼텍스트 마크업 언어(HTML)로 포맷된 웹 페이지입니다. 이 태그 언어는 일반 텍스트, 이미지, 포함된 비디오 및 오디오 콘텐츠, 복잡한 사용자 상호 작용을 구현하는 스크립트(짧은 프로그램)를 지원합니다. HTML 언어는 다른 웹 리소스에 즉시 액세스 할 수있는 하이퍼 링크 (포함 된 URL)도 지원합니다. 웹 탐색 또는 웹 서핑은 여러 웹 사이트에서 이러한 하이퍼링크를 따라가는 일반적인 관행입니다. 웹 응용 프로그램은 응용 프로그램 소프트웨어로 작동하는 웹 페이지입니다. 웹의 정보는 HTTP(하이퍼텍스트 전송 프로토콜)를 사용하여 인터넷을 통해 전송됩니다.
